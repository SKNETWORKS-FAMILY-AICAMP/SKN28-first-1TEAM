# 2. 프로젝트 개요

### 2-1. 프로젝트 주제

**서울시 교통 데이터 분석을 통한 지역별 교통환경 정보 제공**

본 프로젝트는 서울시 자치구별 자동차 등록 대수, 시간대별 교통량, 교통사고 발생 건수 등을 통합 분석하여 지역별 교통환경을 정량적으로 파악하는 것을 목표로 한다.

단순한 수치 제시에 그치지 않고, 차량 등록 대수와 실제 교통량의 차이, 교통 혼잡도와 사고 발생 위험 간의 관계 등을 함께 분석함으로써 각 지역의 교통 특성을 다각적으로 도출하고자 한다.

또한 Python과 MySQL을 활용하여 데이터를 전처리·통합하고, Streamlit 기반의 시각화 서비스를 구현하여 사용자가 지역별 교통 정보를 직관적으로 확인할 수 있도록 한다.

---

## 2-2. 주제 선정 배경

서울시는 지역별로 인구 규모, 상업시설 분포, 도로 구조, 차량 이동 패턴이 다르기 때문에 교통환경 역시 큰 차이를 보인다. 그러나 실제 교통 문제는 단순히 차량 수만으로 설명되지 않으며, 등록 차량 수와 실제 도로 이용량 사이의 차이, 교통량 증가에 따른 사고 위험도 변화 등을 함께 고려해야 보다 현실적인 해석이 가능하다.

---

## 2-3. 프로젝트 목표

본 프로젝트의 주요 목표는 다음과 같다.

- 서울시의 차량 등록 현황, 교통량, 교통사고 데이터를 수집·정제하고, **MySQL 기반의 통합 데이터베이스**를 구축한다.
- 지역별·시간대별 데이터를 활용하여 **교통 혼잡도와 안전도 지표**를 산출한다.
- 데이터 간의 상관관계를 분석하여 **차량 등록 대수, 교통량, 사고 발생 간의 연관성**을 확인한다.
- 사용자가 특정 지역의 교통 혼잡도와 위험도를 한눈에 파악할 수 있도록 **Streamlit 기반 웹 인터페이스**를 구현한다.
- 공공데이터 분석 전 과정을 수행하며 **실무형 데이터 처리 및 시각화 역량**을 강화한다.
- 분산된 공공데이터를 통합·시각화하여 **데이터 기반 교통정보 제공 서비스를 제공** 한다.

---

## 2-4. 기대 효과

본 프로젝트를 통해 다음과 같은 효과를 기대할 수 있다.

- 지역별 차량 등록 대수와 실제 교통량을 비교하여 **도심 교통 유입 및 이동 패턴**을 시각적으로 확인할 수 있다.
- 사고 발생 건수와 교통량을 함께 고려함으로써 **교통 위험도가 높은 지역**을 식별할 수 있다.
- 사용자는 웹 기반 대시보드를 통해 지역별 혼잡도와 안전도를 직관적으로 파악할 수 있어 **교통 정보 활용성**이 높아진다.
- 데이터 기반의 분석 결과는 향후 **교통 관리, 안전 대책 수립, 정책적 판단**을 위한 기초 자료로 활용될 수 있다.
